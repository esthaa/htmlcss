DIVs

The DIV tag on its own doesn't do much.

DIV tag definition

  • The DIV tag has no default styling
  • The DIV tag defines content areas
  • HTML is not funny

DIV tag usage

Assiging an id to a tag allows us to style that tag (and all its nested tags) directly through CSS

To give the id styling it must be named as a selector, along with its dfeclaraction block, and preceded by pound sign (hashtag symbol).

ID usage

The id allows us to identify tags so that they may specifically be adressed within the CSS file.
